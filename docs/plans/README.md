# Active plans

This directory contains concrete work that has been analyzed but is not yet
the current implementation.

Each plan should state:

- status and review date;
- goal and non-goals;
- architecture decision;
- proposed public surface;
- implementation phases;
- test strategy;
- acceptance criteria;
- unresolved decisions.

There are currently no active plans.

When implementation completes:

1. update `../ARCHITECTURE.md`, `../MODULES.md`, and
   `../IMPLEMENTATION_STATUS.md`;
2. update the root README;
3. move the plan to `../archive/` if its historical detail remains useful.
